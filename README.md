# Blockchain Intelligence Research

Forensic on-chain analysis combining transaction tracking, psychological profiling, and algorithmic detection to understand manipulation patterns in crypto markets.

## Focus Areas

- **On-Chain Forensics**: Wallet clustering, fund flow analysis, MEV tracking
- **Behavioral Analysis**: Cognitive bias exploitation, manipulation patterns
- **Algorithmic Tools**: Detection systems, pattern recognition

## Stack

**Analysis**: Arkham Intelligence, Cielo, Dune Analytics, TRM, Chainalysis, Zapper...  
**Development**: Rust, Solidity, C/C++
**Research**: Cognitive biases, social engineering, market psychology

## Scope

Currently focused on memecoin ecosystems as a laboratory for rapid-cycle market manipulation and participant behavior analysis.
